# LITE Studio

LITE Studio هو تطبيق ويب مستقل لتخطيط إعدادات استوديو التصوير على الكمبيوتر والجوال.

يساعدك على بناء المخطط بسرعة، وقياس المسافات المهمة داخل البلاتوه، والتكبير داخل المخطط، وحفظ إعداداتك الخاصة، وتصدير النتيجة بصيغة PNG أو PDF.

## المميزات

- واجهة responsive مناسبة للجوال والكمبيوتر.
- منظر علوي مع تكبير، وعجلة ماوس على الكمبيوتر، و pinch-to-zoom على الجوال، ومعاينة أيزومترية قابلة للتكبير.
- مكتبة عناصر مع بحث حسب النوع أو المقاس أو الفئة.
- قياسات تلقائية بين الهدف والكاميرا والخلفية ومصادر الضوء، بالإضافة إلى وضع قياس يدوي.
- إعدادات للعناصر ومعلومات عملية مثل الارتفاع والاتجاه وقدرة الضوء وقراءة f-stop تقديرية عند ISO 100.
- إعدادات جاهزة مدمجة وإعدادات مخصصة محفوظة داخل المتصفح.
- تصدير PNG وتصدير PDF وطباعة مباشرة للمخطط الرئيسي.
- واجهة متعددة اللغات: الفرنسية والإنجليزية والإسبانية والألمانية والعربية.

## البدء السريع

لا حاجة إلى أي تثبيت.

1. افتح [app.html](app.html) في متصفح حديث لتشغيل التطبيق، أو [index.html](index.html) لعرض صفحة الهبوط العامة.
2. أضف العناصر من المكتبة.
3. حرّكها داخل المخطط ودوّرها باستخدام المقبض البرتقالي.
4. استخدم عجلة الماوس على الكمبيوتر أو حركة القرص على الجوال للتكبير داخل المنظر العلوي.
5. فعّل `قياس` لعرض المسافات حول الهدف أو لرسم قياس يدوي.
6. استخدم قائمة `الإعدادات` لتحميل إعداد جاهز أو حفظ إعدادك الخاص.
7. استخدم قائمة `ملف` لنسخ الصورة أو تصدير PNG أو PDF أو طباعة المخطط.

## بنية المشروع

- `index.html`: صفحة الهبوط العامة.
- `app.html`: تطبيق تخطيط استوديو التصوير.
- `css/simple-studio.css`: التصميم والـ responsive والتخطيط وأنماط الطباعة.
- `css/site.css`: أنماط صفحة الهبوط والصفحات التحريرية.
- `js/simple-studio.js`: منطق الاستوديو ورسم الـ canvas واللغات والإعدادات الجاهزة والقياسات والتفاعلات.
- `js/site-i18n.js` و `js/site-copy-*.js`: محتوى الصفحات العامة وتوطينها.
- `guide.html` و `presets.html` و `faq.html`: صفحات شرح ومرافقة.
- `schema-eclairage-portrait.html` و `plan-studio-boudoir.html` و `lighting-diagram-interview.html`: صفحات SEO مرتبطة بالتطبيق.
- `docs/screenshots/`: لقطات الشاشة المستخدمة في README والمعاينات.

## معاملات URL مفيدة

يمكن استخدام معاملات بسيطة لفتح لغة أو إعداد جاهز مباشرة:

- `?lang=fr`
- `?lang=en`
- `?lang=es`
- `?lang=de`
- `?lang=ar`
- `?preset=portrait-soft-45`
- `?preset=boudoir-parabolic`
- `?preset=interview-led`

مثال:

```text
index.html?preset=portrait-soft-45&lang=ar
```
