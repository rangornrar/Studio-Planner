Guide

Comment preparer un plan de studio clair, exploitable et facile a transmettre a l'equipe.

Le but n'est pas de dessiner un schema abstrait. Il s'agit de placer sujet, camera, source cle, fond et accessoires de controle de facon lisible, avec assez d'informations pour monter le plateau sans hesitation.

1. Poser la base

Placez le fond, le sujet et l'axe camera avant le reste.

Commencez par les reperes qui structurent tout le plan : fond, sujet, orientation de prise de vue. Une fois cette base claire, les sources et accessoires trouvent leur place beaucoup plus vite.

  • Choisir un fond coherent avec l'intention
  • Garder un axe lisible entre sujet et camera

2. Installer la source cle

Ajoutez la lumiere principale avant de multiplier les accessoires.

Une grande source bien placee couvre deja beaucoup de cas. Commencez par elle, puis ajoutez reflecteurs, flags ou lumieres secondaires seulement si le rendu ou la circulation le demande.

  • Portrait doux : source large a 45 degres
  • Interview : axe camera clair et panneau principal simple

3. Verifier les distances utiles

Activez Mesure une fois la base en place.

Le mode Mesure affiche rapidement les distances sujet camera fond et sources. Cela permet de valider un setup avant de le monter et de parler avec des chiffres plutot qu'avec des impressions.

  • Comparer camera sujet et sujet fond
  • Verifier la distance de la source principale

4. Zoomer pour relire le plateau

Controlez circulation, collisions et lisibilite sans perdre le cadre general.

La molette sur desktop et le pinch sur tablette ou mobile aident a verifier les pieds, le mobilier, les degagements et les details qui comptent avant d'envoyer le plan.

5. Sauvegarder une version de travail

Transformez une base en preset adapte a votre studio.

Chargez un preset, adaptez-le a votre espace, a votre sujet ou a votre client, puis sauvegardez votre propre variante dans le navigateur pour les prochains shootings.

6. Partager le plan final

Copie d'image, PNG, PDF ou impression selon le besoin.

Une fois le setup valide, choisissez la sortie la plus utile pour le brief, la feuille de service, le message equipe ou la pre production.
